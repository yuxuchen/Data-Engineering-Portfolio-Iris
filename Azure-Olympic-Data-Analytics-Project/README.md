# End-to-End Azure Olympic Data Analytics Project
## Introduction
This project focuses on building a complete data pipeline for analyzing Paris Olympics data using Microsoft Azure. It integrates multiple Azure services to extract, transform, store, and visualize data efficiently.

## Architecture
![project architecture](projectArchitecture.jpg)

## Technology Used
1. Azure Data Factory: Orchestrates data extraction and loading.
2. Azure Data Lake Gen2: Stores raw and transformed data for further processing.
3. Databricks: Handles data transformation and processing.
4. Azure Synapse Analytics: Provides a data warehouse for analytical queries.
5. Power BI: Visualizes insights with an interactive dashboard.

## Dataset Used
The dataset from Kaggle contains information about athletes, medals, sports disciplines, events, and other related details for the Paris 2024 Olympics.

Here is the dataset link:  https://www.kaggle.com/datasets/piterfm/paris-2024-olympic-summer-games
